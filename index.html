O sistema operacional desenvolvido por um sonhador de 13 anos!

Sobre

Oi, meu nome é Matheus, tenho 13 anos e sempre gostei de programar desde os 8. Aos 10, comecei a usar Linux e percebi como sistemas operacionais (OS) são fascinantes. Por trás de toda aparência e funcionalidade, eu me perguntava: "Será que é difícil fazer um?"

Foi aí que descobri o quão complicado é criar um OS do zero. Pensei em desistir, mas ao ver projetos no OSdev Wiki, fiquei super motivado a tentar mais uma vez. Hoje, estudo todos os dias para aprimorar o MateeOS!

Exemplo de inicialização


void entry() {
    print("Olá mundo!\n");
    halt_sys();
}

Saída:


Olá mundo!

System halted.

Observação

Não se preocupe em incluir arquivos como #include, já que isso é feito automaticamente no momento da compilação. O compilador organiza tudo para você!

Exemplo de função de inicialização no MateeOS


void init() {
    print("Iniciando MateeOS...\n");
    // Configurações de sistema e drivers
    setup_drivers();
    load_filesystem();
    print("Sistema pronto para uso!\n");
    halt_sys();
}

Saída:


Iniciando MateeOS...

Drivers carregados.
Sistema pronto para uso!

System halted.
